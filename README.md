# docker-scala

Dockerfile for installing Scala 2.11.1/2.10.4 and Java 7 on Ubuntu 14.04 (phusion/baseimage)

## Installation

```
docker pull shouldbee/docker-scala
```

## Contribution

### Build images

```
make build
```

### Test images

```
make test
```

### Push to Docker registry hub

```
make push
```
